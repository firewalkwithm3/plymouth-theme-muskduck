## Musk Duck Plymouth Theme

INSTALL

    1. Move to /usr/share/plymouth/theme/muskduck or whatever.
    2. Edit 'muskduck.plymouth' and replace @IMAGES@ with that path.
    3. $ plymouth-set-default-theme -R muskduck
